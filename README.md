# AI-based Log Anomaly Detection System

## Overview
This project is an end-to-end AI and MLOps system for detecting anomalies in logs using machine learning.

## Goals
- Parse and preprocess logs
- Extract useful features
- Detect anomalies with ML
- Expose predictions through an API
- Add testing, security checks, and deployment support

## Tech Stack
- Python
- FastAPI
- scikit-learn
- Docker
- Kubernetes
- pytest
- bandit
- pip-audit
